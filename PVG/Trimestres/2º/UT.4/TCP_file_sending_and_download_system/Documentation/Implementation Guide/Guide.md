 ##### (It is **recommended** to use two terminals simultaneously for better performance during program execution)



1. Navigate to the program's main folder:
   	**cd** .\Downloads\TCP_file_sending_and_download_system\src\main\java
   
2. Run the compilation commands:
   	Terminal 1:
   		**javac** Server\\FileServer.java
   	terminal 2:
   		**javac** Client\\FileClient.java
   
3. Launch the execution of the file (the class, ".java"):
   	Terminal 1:
   		**java** Server.FileServer
   	terminal 2:
   		**java** Client.FileClient
   
4. If everything has gone correctly, the "Client" will ask you for a file name; additionally, if you have downloaded the entire project, you will find a text file "Test.txt" in the "Server/" > "files/" folder.
